Was ist MyBudget?

MyBudget ist eine simple Software zum Verwalten von Haushaltseinnahmen und - ausgaben. MyBudget ermöglicht dir den Überblick über deine Finanzen zu behalten und zu verfolgen, wofür du dein Geld ausgiebst.

Was kann MyBudget?

MyBudget kann Einnahmen und Ausgaben beliebiger Kategorien wie z.B. Essen, Getränke, Sportverein, Auto, etc. erfassen und sortieren. Du kannst mit MyBudget dein Kaufverhalten analysieren und dir anzeigen lassen, wie sich deine Ausgaben in einzelnen Kategorien entwickeln. Darüber hinaus kannst du in der MyBudget-Datenbank auch nach Jahren noch gezielt nach bestimmten Einnahmen und Ausgaben suchen. So findest du z.B. heraus, wann du deinen DVD-Player gekauft hast.

Installation

MyBudget muss nicht installiert werden. Es reicht die Datei MyBudget.jar zu starten. Dies geht entweder über eine Doppelklick auf ide Datei oder über die Komandozeile via java -jar MyBudget.jar

MyBudget legt beim ersten Start zusätzlich benötigte Dateien automatisch an.

Systemvorraussetzungen

Zum Ausführen von myBudget wird Java benötigt. Das Java RUntime Enviroment (JRE) kann von der Java-Website herunter www.java.com heruntergeladen werden.

MyBudget Dateien

MyBudget speichert Fehlermeldungen in Log-Dateien im unterverzeichnis log. Die Datenbank befindet sich in der Datei mybudget.db. Die Datenbank kann beliebig portiert werden. Beim Start von MyBudget wird überprüft ob die Datenbank vorhanden ist und diese ggf. neu erstellt. Um die Datenbank komplett zu löschen genügt es die Datei mybudget.db zu löschen. Einstellungen von MyBudget werden in der Datei mybudget.properties gespeichert. Diese sollte in der Regel nicht verändert werden!

Update

MyBudget prüft beim Start ob eine neue Programmversion Verfügbar ist und läd diese automatisch herunter. Die Daten in der Datenbank gehen dabie nicht verloren.

Fehlerbehung

MyBudget befindet sich zur Zeit noch im Feldversuch. Sollten Probleme auftreten, wenden Sie sich mail@private-factory.de. Geben Sie bitte eine möglichst detaillierte Beschreibung des Fehlers an und senden Sie die Log-Dateien aus dem Ordner log für den entsprechenden Tag, an dem der Fehler aufgetreten ist, sowie die Dateien mybudget.db und mybudget.properties mit.
